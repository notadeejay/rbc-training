### What colour will the paragraph text be?

```html
  <header>
    <h1>My Awesome Webpage</h1>
    <p id="subtitle">I'm a paragraph</p>
  </header>

```

```css
  #subtitle {
    color: purple;
  }
  header p {
    color: green;
  }
```

### What font-size will the h4 element be?

```html
  <main class="news-articles">
    <h4>What font-size will I be?</h4>
    <p>Paragraph</p>
  </main>
```

```css
  main h4 {
    font-size: 40px;
  }

  .news-article h4 {
    font-size: 30px;
  }
```
